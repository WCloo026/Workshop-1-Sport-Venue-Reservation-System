# Workshop-1--Sport-Venue-Reservation-System
## Project Overview
The Sports Venue Reservation System (SVRS) is a robust Command Line Interface (CLI) application developed as a core module of the Computerized Management Information System (CMIS). Designed for sports facility rental companies, the system streamlines the end-to-end process of venue management, from real-time availability tracking to automated financial auditing.

## Objectives
1. To provide an easy-to-use platform for booking sports venues
2. To manage venue information efficiently
3. To automate booking and price calculation
4. To prevent double booking of time slots
